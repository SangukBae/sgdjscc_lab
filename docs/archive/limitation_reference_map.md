# SGD-JSCC 한계점 우선순위 지도

> 보관본. 최신 요약은 [etri_strategy.md](../etri_strategy.md)를 우선 참조.

ETRI 과제 초점은 최대 `PSNR`이 아니라 **무선 전송 후 시맨틱 의도의 신뢰성 있는
보존**이다. 이 기준에서 `sgdjscc_lab`이 우선 해결해야 할 핵심 한계를 3가지로 정리한다.

## 핵심 한계점

| 한계 | 내용 | `sgdjscc_lab` 대응 |
|---|---|---|
| **1** | 화질은 자연스럽지만 없던 객체·정보가 생성될 수 있고, 이를 측정할 지표가 없음 | 할루시네이션 탐지, 객체 추가·누락·왜곡 검출, packet-aware verifier, VQA 기반 평가, regeneration |
| **2** | 평가가 `PSNR`·`SSIM` 같은 화질 중심이고 의미 보존 지표가 부족함 | SRS/srs_packet/srs_v2, 의미 유사도, 객체 보존율, 누락률, 추가 객체율, 의미 충실도·신뢰성 평가 체계 |
| **3** | 정지 이미지 중심이고 영상의 시간 흐름과 장면 전환을 다루지 못함 | keyframe 기반 비디오 파이프라인, scene change 처리, temporal consistency 평가, temporal SRS |

1번이 ETRI가 가장 강조한 수신부 생성형 AI 할루시네이션 완화와 직접 맞닿아 있고,
2번이 그 문제를 정량 평가하는 기반이며, 3번이 정지 이미지에서 영상으로 확장하는 핵심 축이다.

## 우선순위 근거

- **1**: 수신단 생성형 AI가 그럴듯하지만 틀린 객체를 추가하거나, 원래 있어야 할 정보를
  누락·왜곡하면 시맨틱 통신의 목표를 직접 위반한다. 따라서 검출과 완화가 최우선이다.
- **2**: 1번 문제를 해결했다는 주장을 하려면 `PSNR`만으로는 부족하다. 송신 의도와 수신
  복원이 얼마나 일치하는지 평가하는 의미 충실도·신뢰성 지표가 필요하다.
- **3**: ETRI 과제는 정지 이미지 복원에 머물지 않고 keyframe 기반 영상, 장면 전환,
  시간적 의미 일관성까지 검토 대상으로 둔다. 따라서 시간축 대응이 핵심 확장이다.

## 보조 연구 축

- 시맨틱 부가정보의 견고성·오버헤드
- 확산 복원 지연과 few-step/consistency 가속
- 페이딩·블라인드 채널 적응
- `128×128` 중심 실험 제약
- MIMO/OFDM/다중 사용자 확장

## 워크스트림 매핑

| 워크스트림 | 계획 역할 |
|---|---|
| 할루시네이션 완화·검증 | 객체 추가·누락·왜곡 검출, packet-aware verifier, VQA, regeneration |
| 의미 충실도 평가 체계 | SRS 계열, 객체 보존율, 누락률, 추가 객체율, 의미 일치 기반 최종 선택 |
| 시간축 확장 | keyframe/scene change 처리, temporal evaluator, temporal SRS |
| 보조 최적화 | 가이드 손상 모델, 저지연 샘플링, 채널 조건화, 이후 MIMO/OFDM |
